# Expense Tracker in Python 💰

This is a simple **Expense Tracker project built using Python**.
It helps users record and manage their daily expenses.

## Features

* Add new expenses
* View recorded expenses
* Simple and beginner-friendly program

## Technologies Used

* Python

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the Python file:

```
python main.py
```

## Purpose

This project was created to practice **Python programming and logic building**.
